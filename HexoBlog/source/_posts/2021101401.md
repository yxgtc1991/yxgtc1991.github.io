# 二、搭建Hexo博客
### 1、安装Hexo
```
npm install -g hexo-cli
```
### 2、Hexo初始化
```
hexo init      # 初始化
npm install    # 安装组件
```
{% note info %}
Hexo博客目录结构：</br>
{% endnote %}
```
|-- public			# 网站文件
|-- scaffolds		# 模板
|-- source			# 用户资源
     |-- _post		# markdown文档
|-- themes			# 主题
|-- _config.yml		# 网站配置信息
|-- package.jaon	# 应用程序信息
```
### 3、启动本地服务器预览
```
hexo g		# 生成页面
hexo s    	# 启动预览[支持本地热更新]
```
### 4、访问localhost:4000，出现Hexo默认页面，安装成功
{% asset_img 5.png %}</br></br>
# 三、部署Hexo到Github Pages
### 1、安装hexo-deployer-git
```
npm install hexo-deployer-git --save
```
### 2、修改_config.yml的deploy部分
```
deploy:
  type: git
  repository: https://github.com/用户名/用户名.github.io
  branch: 分支名
```
### 3、将Hexo博客上传部署到Github Pages
```
hexo d
```
### 4、访问Github域名，出现Hexo博客页面，部署完成
# 四、发布文章
### 1、创建文章
```
hexo new 文章名
```
### 2、部署到Github Pages
```
hexo g		# 生成页面
hexo d    	# 部署发布
```
